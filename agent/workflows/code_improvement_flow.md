description: Standardize how to improve a file's quality (structure and documentation).
---

1. Identify the target file.
2. Analyze and Improve Structure.
   - Call skill: `review_file_structure`.
   - Apply strict improvements if agreed.
3. specific: Update Documentation.
   - Call skill: `update_comments`.
   - Ensure docstrings and comments match the *new* structure.
4. Verify Integerity.
   - Ensure the code still behaves as expected (no breaking changes).
